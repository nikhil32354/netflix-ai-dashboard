# 🎬 Netflix AI Data Analytics Dashboard 🤖

An AI-powered data analytics dashboard with a natural language chatbot that allows users to explore Netflix data using simple English queries.

---

## 🚀 Features

- 📊 Interactive Dashboard (Charts & Insights)
- 🤖 AI Chatbot (Ask questions in plain English)
- 📂 Upload CSV dataset
- 📈 Auto-generated visualizations
- ⚡ Real-time data analysis using PandasAI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- PandasAI
- OpenAI API

---

## 📊 Example Questions

- "How many movies are there?"
- "Top 5 genres?"
- "Shows released after 2015?"
- "Which country produces most content?"

---

```bash
pip install -r requirements.txt
